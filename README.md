# ねこレコ

**多頭飼育の個体管理アプリ**

ねこレコは、たくさんの“うちの子”を、ひとりずつ大切に記録・管理するためのアプリです。  
猫ごとのプロフィール、健康記録、通院予定、ワクチン、駆虫薬、フード、保険、家族共有などをまとめて管理できます。

---

## 開発環境

アプリ本体は `src/` 配下の Expo + React Native + TypeScript プロジェクトです。

### 前提

- Bun
- iOS: Xcode と iOS Simulator、または Expo Go を入れた実機
- Android: Android Studio と Android Emulator、または Expo Go を入れた実機

### セットアップ

```bash
cd src
bun install
```

### 起動

```bash
cd src
bun run start
```

iOS で起動:

```bash
cd src
bun run ios
```

Android で起動:

```bash
cd src
bun run android
```

### 検証

```bash
cd src
bun run lint
bun run build
bun run test
```

Phase 0 では、MVP 非対象機能を作り込まず、後続のオンボーディング、ホーム、猫一覧、記録、通知を追加できる基盤だけを配置しています。

## コンセプト

**たくさんの“うちの子”を、ひとりずつ大切に記録・管理できるアプリ。**

多頭飼いにおける管理の大変さは、単一の課題ではなく、「個体ごとの情報が多く、かつ家族内で共有しづらい」ことにある。本アプリは、猫ごとの医療・食事・生活・性格・費用情報を一元管理し、家族で共有できる個体管理基盤を提供する。

---

## ターゲットユーザー

### メインターゲット

猫を複数匹飼っている一般家庭。

特に、3匹以上の猫と暮らしていて、健康管理・ごはん・通院・性格・投薬・保険・記念日などの情報が、メモアプリ、写真フォルダ、カレンダー、家族の記憶、紙の書類などに分散している人を想定しています。

### 将来的な拡張ターゲット

- 保護猫活動者
- 一時預かりボランティア
- 譲渡活動を行う個人・団体
- ペットシッター・家族以外のお世話担当者

---

## 解決したい課題

多頭飼育では、1匹ごとに必要な情報が多くなります。

- ワクチンや駆虫薬の接種日を忘れてしまう
- 次回の接種日や通院予定を忘れてしまう
- 偏食の子が食べてくれた貴重なフードの銘柄を忘れてしまう
- 留守中のお世話を家族に頼むとき、注意事項の共有が大変
- 誕生日やうちの子記念日を忘れてしまう
- ペット保険の請求が済んでいるか分からなくなる
- 病院で「次回は○週間後」と言われたのに忘れてしまう
- 診察室で測った体重をあとで記録しようとして忘れてしまう
- 猫ごとの情報が増えるほど、どの子の情報か分かりづらくなる

ねこレコは、これらの情報を猫ごとに整理し、記録・通知・共有までできるようにすることで、多頭飼い家庭の不安と手間を減らします。

## 主な機能

### 猫プロフィール管理

猫ごとの基本情報を登録します。

- 名前
- 写真
- 生年月日
- 性別
- うちの子記念日
- 猫種・血統
- 毛色・柄
- 性格
- 注意事項
- マイクロチップ番号
- 迎えた経緯

### 医療・予防管理

通院や予防医療の記録を管理します。

- 通院履歴
- ワクチン接種日
- 次回ワクチン予定
- 駆虫薬の投与日
- 次回駆虫薬予定
- 検査履歴
- 避妊・去勢情報
- 病歴
- 主治医・かかりつけ病院
- 次回通院予定

### 通知・リマインダー

忘れやすい予定を事前に通知します。

- ワクチン予定
- 駆虫薬予定
- 通院予定
- 投薬予定
- 誕生日
- うちの子記念日
- 保険請求の未対応通知

### フード管理

猫ごとの食事情報を記録します。

- いつものフード
- 食べたフード
- 食べなかったフード
- 好きな味・形状
- 苦手なごはん
- アレルギー
- 食後の体調メモ

### 体重・体調記録

日々の変化を猫ごとに残します。

- 体重記録
- 体調メモ
- 嘔吐
- 下痢・軟便
- 食欲
- 飲水
- 排泄
- ケア記録

### 保険・費用管理

通院費や保険請求の状態を管理します。

- 通院日
- 病院名
- 診療費
- 保険対象かどうか
- 請求ステータス
  - 未請求
  - 請求準備中
  - 請求済み
  - 入金済み
  - 対象外
- 領収書写真
- 診断名
- メモ

### 家族共有

家族やお世話担当者と情報を共有します。

- 家族メンバー招待
- 閲覧・編集権限
- 今日のお世話
- 投薬済みチェック
- 留守中の注意事項
- 猫ごとの申し送り
- 緊急時情報

## 初回オンボーディング

ねこレコの初回オンボーディングは、入力負担を抑えながら、猫ごとの個体管理に必要な土台を作る設計です。

### 1. スプラッシュ画面

ロゴを表示します。

> たくさんの“うちの子”を、ひとりずつ大切に記録。

### 2. ウェルカム画面

ねこレコでできることを簡潔に伝えます。

- 猫ごとの情報をまとめて管理
- 通院・ワクチン・駆虫薬を通知
- 家族とお世話情報を共有

### 3. 最初の猫ちゃん登録

初回登録では、プロフィール作成に必要な項目に絞って入力します。

- 写真
- 名前
- 生年月日
- 性別
- うちの子記念日
- 猫種・血統
- 毛色・柄

生年月日や血統が不明な猫も登録しやすいように、「不明」「推定」「雑種」「毛色のみ」といった選択肢を用意します。

### 4. 登録完了後の分岐

猫の登録完了後、次の2つの導線を表示します。

#### 詳しい情報も入力する

ワクチン、駆虫薬、主治医、ペット保険、病歴、フード、注意事項などを続けて登録できます。

#### あとで入力する

すぐにホーム画面または猫一覧画面へ移動し、日常利用を開始できます。  
詳細情報は、猫のプロフィール画面から後でいつでも追加できます。

## 画面設計の基本方針

ねこレコは、猫ごとの情報量が多くなりやすいアプリです。  
そのため、画面構成はシンプルで分かりやすく、初回登録以降は少ない操作で使える設計にします。

### 設計思想

- 初回登録は迷わせない
- 詳細管理は押しつけない
- 情報は深く持てる
- 操作は浅く使える
- 今日必要なことだけがすぐ分かる
- 多頭飼いでも、1匹ずつ大切に管理できる

## ブランドイメージ

**かわいいけど、ちゃんと管理できる。**

ねこレコは、単なる猫の日記アプリではありません。
